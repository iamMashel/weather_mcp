# Security Policy

## Supported Versions

The `main` branch is the actively supported version.

## Reporting a Vulnerability

Please do not open a public issue for sensitive security reports. Instead, contact the maintainer privately with:

- A clear description of the issue
- Steps to reproduce
- Impact and affected versions
- Any suggested fix, if available

## Data Handling

Weather MCP does not require API keys for National Weather Service data and does not store user accounts, secrets, or personal weather history.
